# 用户使用文档

## 快速开始
- 注册或使用第三方登录
- 创建研究项目并依次完成模块
- 在仪表盘查看进度与用量

## 功能说明
- 注册与登录：支持邮箱密码、Google、GitHub
- 创建项目：填写标题与领域，进入模块流程
- 模块使用：
  - 选题生成：输入关键词与描述，生成6个候选并评分
  - 理论框架：依据选题生成研究框架与热点研究
  - 文献综述：生成大纲与章节，导出HTML或Word
  - 研究设计：生成方法、材料与行动计划
  - 论文组装：整合章节生成最终HTML
- 订阅管理：查看当前计划与使用限制

## 常见问题
- 生成超时：前端会显示提示并后台继续处理，稍后刷新或在页面自动轮询获取结果
- 限流提示：稍后重试或升级订阅
- 余额不足：联系管理员
